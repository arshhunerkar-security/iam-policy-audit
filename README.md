# IAM Policy Audit Simulator — TechNova Inc.

## Project Overview
A simulated Identity and Access Management (IAM) audit for a fictional company, TechNova Inc.
This project demonstrates core IAM concepts including Least Privilege, RBAC, Segregation of Duties,
and the identification of orphaned accounts and privilege creep.

## Files
- `data/user-access-register.xlsx` — Full employee access register
- `data/audit-findings.xlsx` — Flagged violations with risk ratings
- `report/TechNova-IAM-Audit-Report.pdf` — Executive audit report with recommendations

## Skills Demonstrated
Identity & Access Management | Access Control Policy | Risk Assessment | Security Reporting

## Disclaimer
All data in this project is entirely fictional and created for portfolio purposes only.
